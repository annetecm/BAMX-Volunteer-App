# BAMX Volunteer App 🍎🤝

Una aplicación móvil diseñada para optimizar y gestionar la participación de voluntarios en el ecosistema del **Banco de Alimentos de México (BAMX)**. Esta herramienta facilita la coordinación entre la organización y los voluntarios.

## 📋 Descripción

La **BAMX Volunteer App** centraliza las operaciones de voluntariado, permitiendo que los usuarios descubran oportunidades, se registren en las diferentes tareas disponibles y lleven un control de su impacto social. El objetivo es digitalizar procesos que antes eran manuales para aumentar la eficiencia del banco.

## ✨ Características Principales

*   **Registro de Voluntarios:** Perfiles personalizados con intereses y habilidades.
*   **Catálogo de Actividades:** Visualización de eventos disponibles (colectas, clasificación de alimentos, administración).
*   **Check-in Digital:** Registro de asistencia mediante la aplicación.
*   **Historial de Impacto:** Los voluntarios pueden ver cuántas horas han contribuido y el impacto estimado de su labor.
*   **Notificaciones:** Alertas sobre nuevas convocatorias o cambios de último momento en las brigadas.

## 🛠️ Tecnologías Utilizadas

Este proyecto utiliza un stack moderno para garantizar escalabilidad y rendimiento:

| Tecnología | Propósito |
| :--- | :--- |
| **React Native** | Framework para el desarrollo móvil multiplataforma. |
| **Firebase** | Autenticación, base de datos en tiempo real y notificaciones push. |
| **Node.js** | Backend para lógica de negocio compleja. |
| **Git** | Control de versiones y colaboración. |

## 🚀 Instalación y Configuración

Para ejecutar este proyecto localmente, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/annetecm/BAMX-Volunteer-App.git](https://github.com/annetecm/BAMX-Volunteer-App.git)
    ```

2.  **Instalar dependencias:**
    ```bash
    npm install  
    
    ```

3.  **Configurar variables de entorno:**
    Crea un archivo `.env` o configura el archivo de servicios de Firebase (`google-services.json` o `GoogleService-Info.plist`) en la carpeta correspondiente.

4.  **Ejecutar la app:**
    ```bash
    npm start
    ```
